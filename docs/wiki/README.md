## 关于开源协议
蓝鲸日志采集器采用的是 MIT 开源协议。MIT 是和 BSD 一样宽泛的许可协议，作者只想保留版权，而无任何其他的限制。也就是说，你必须在你的发行版里包含原许可协议的声明。
